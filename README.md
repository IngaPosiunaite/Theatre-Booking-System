# Theatre Booking System


## Introduction

The Theatre Booking System is a comprehensive platform designed to streamline the process of booking tickets for performances at The Theatre Royal. 
This system provides users with easy access to information about upcoming shows and enables them to conveniently purchase tickets online.

## Usage / Installation 

To run the Theatre Booking System locally, follow these steps:

Clone the repository by running the following command inside a terminal

```
git clone git clone https://github.com/your-username/theatre-booking-system.git
```

Then within the folder, run 

```
java -jar theatre-booking-system.jar
```

## Features

1. Browse Shows/Performances: Users can browse through a list of upcoming shows and performances at The Theatre Royal.
2. Search Functionality: The system allows users to search for specific shows or performances based on show titles or dates.
3. Ticket Booking: Users can select the number of tickets they wish to purchase for a particular show and proceed to complete the ticket purchase process.
4. Secure Transactions: The system ensures secure transactions for ticket purchases, maintaining the confidentiality of user data.
5. User-Friendly Interface: The platform offers a user-friendly interface, making it easy for users to navigate and interact with the booking system.

## Project Tasks

The development of the Theatre Booking System involved the following tasks:

1. Database Design and Implementation: Designing and implementing the underlying database structure to store information about shows, performances, and user transactions.
2. Back-End Model Development: Developing the back-end model for managing shows, performances, and user interactions.
3. Class Diagram Creation: Creating a comprehensive class diagram based on project specifications to visualize the relationships between different classes.
4. Unit Testing: Developing unit tests for Java classes to ensure the reliability and functionality of the back-end model.
5. Integration with Front-End Interface: Analyzing and implementing the integration of the back-end model with the front-end interface, utilizing the JDBC API for seamless communication between components.
6. Data Analysis Report: Producing a detailed data analysis report regarding the usage of the website and The Theatre Royal's utilization of the booking system.
