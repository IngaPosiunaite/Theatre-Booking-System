<h3> Theatre Booking System


<h5> The project's objective was to develop a new booking system for The Theatre Royal, which provides information about upcoming shows and enables people to purchase performance tickets online.

<h5> The system allows users to:

<h5>• Browse shows/performances.</h5>
<h5>• Search for specific shows or dates.</h5>
<h5>• Select a number of tickets for a show for purchase.</h5>
<h5>• Complete their ticket purchase.</h5>

<h3> High-level Tasks

<h5> The following tasks were undertaken:

<h5> • Browse shows/performances.</h5>
<h5>• The design and implementation of the underlying database.</h5>
<h5>• The design and implementation of the back-end model for shows/performances.</h5>
<h5>• The creation of a class diagram based on the project specification and the relationships between the classes.</h5>
<h5>• The development of unit tests for Java classes.</h5>
<h5>• The analysis of how the back-end model integrates with the front-end interface, including the use of the JDBC API in implementation.</h5>
<h5>• The production of a data analysis report concerning the website and The Theatre Royal's utilization of it.</h5>
